## Этап 2: Code Review и исправление проблем (task2)

**Цель:** Аудит кода, сгенерированного в task1 — найти и исправить минимум 5 проблем.

### Промпт 1
**Инструмент:** Claude Code  
**Запрос:** > 
«Проведи строгий аудит кода, который ты сгенерировал в предыдущем шаге.
Найди и исправь не менее 5 проблем из следующих категорий:
1. Уязвимости безопасности (например, SQL-инъекции или отсутствие валидации прав в JWT).
2. Проблемы производительности (например, N+1 запросы в SQLAlchemy).
3. Нарушения стиля (PEP8) и отсутствие обработки специфичных исключений БД.
Выведи результат в виде таблицы: 'Что сгенерировал ИИ' → 'В чем проблема' → 'Как исправил'.»

**Результат:**

Найдено и исправлено **7 проблем**:

| # | Категория | Проблема | Исправление |
|---|-----------|----------|-------------|
| 1 | **Безопасность** — тайминг-атака | `if user is None or not verify_password(...)` — bcrypt не вызывается для несуществующих email → можно перебирать адреса по времени ответа | Всегда вызывать `verify_password` с реальным или `_DUMMY_HASH` |
| 2 | **Безопасность** — слабый дефолт | `SECRET_KEY: str = "change-me-..."` — приложение стартует с известным секретом без `.env` | `Field(..., min_length=32)` — Pydantic падает при старте без ключа |
| 3 | **Производительность** — лишние round-trips | `db.refresh(workout)` + сразу `SELECT` с `selectinload` — 2 лишних запроса к БД | Убраны `db.refresh` в `create_workout` и `update_workout` |
| 4 | **Производительность** — нет пагинации | Все list-эндпоинты без `skip`/`limit` → OOM и таймауты при тысячах записей | `skip/limit` query-параметры на все 4 list-эндпоинта |
| 5 | **Типизация (PEP 484)** | `async def get_db() -> AsyncSession` — функция-генератор, тип неверный | Исправлено на `AsyncGenerator[AsyncSession, None]` |
| 6 | **Обработка исключений БД** | Нет `except IntegrityError` в `exercises.py`, `progress.py` → HTTP 500 при race condition | `try/except IntegrityError` → rollback → HTTP 404/422 |
| 7 | **Стиль (PEP 8/484)** | `def do_run_migrations(connection)` без аннотаций в `alembic/env.py` | Добавлено `(connection: AsyncConnection) -> None` |

Таблица исправлений сохранена в `task2/REVIEW.md`.
